**MeetOUT** is a social networking platform focused on events, designed to assist users in discovering activities tailored to their interests within their vicinity. Whether you are exploring a new city or looking for local engagement, MeetOUT simplifies the process of creating and joining events, fostering community engagement and social interaction.

## ğŸŒŸ Key Features

* **Event Discovery:** Easily discover events nearby through a user-friendly interface.
* **Create and Join:** Registered users can organize new events or join activities created by others in the community.
* **Automatic Group Chats:** The application automatically generates group chats for each event, facilitating seamless communication and media sharing among participants.
* **Personalized Profiles:** Each user has a profile to manage their created and joined events, bio, and profile picture.
* **Social Connections:** Search for other users, access their profiles, and manage a friends list to discover even more events.
* **Map & List Views:** Visualize events on an interactive map or switch to a list view for events happening nearby.
* **Responsive UI:** The layout is fully responsive, optimized for both smartphone and tablet experiences.
